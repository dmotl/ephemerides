Basic information

This application computes times of minima based on data stored in off-line catalogue files. It is intended for visual, CCD and PE observers. The program works in MS Windows environment. MS Windows 98 SE or newer is required.

Current installation package includes following catalogue files:

  • BRNO - valid to 11. Dec 2010
  • BRKA 2005 - valid to 28. Feb 2005
  • Prosper - valid to 10. Aug 2006

You can also created your own catalogue files and use it in the program.

All times are displayed in UT!
