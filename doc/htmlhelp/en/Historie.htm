Change log

Version 1.22
  • A new option to filter records by object-Moon distance
  • A new option to copy some fields for selected row to clipboard
Version 1.21
  • An option to restore last used date
Version 1.20
  • A new column PS added to Star ephemerides
  • Merged columns Name and Cons on Daily ephemerides
  • Save current sort mode
  • Highlight current mode in the sort menu
Version 1.19
  • Fixed bug - swapped magnitude in minimum between a primary and secondary minimum.
Version 1.18
  • The first column split into constellation, variable name and minimum type.
  • Added sorting by constellation and minimum type as options.
Version 1.17
  • Added context menu on the list of catalogue files
  • The BRNO catalog was updated from the on-line database (http://var.astro.cz/)
Version 1.16
  • New filtering rule - by variability type
  • The BRNO catalog was updated from the on-line database (http://var.astro.cz/), the variability types were taken from GCVS
Version 1.15
  • Added new columns to tables - Lunar disk illumination in percents and object-to-Moon distance in degrees.
  • The 'Moon' dialog shows Lunar phase in days and Lunar disk illumination in percents.
  • The BRNO catalog was updated from the on-line database (http://var.astro.cz/)
Version 1.14
  • Number of records and number of variable stars is shown in the status bar
Version 1.13
  • Updated 'Brightness in minimum' - shows one more decimal place
Version 1.12
  • Fixed issues related to time zones
  • New filtering rule - list of constellations
Version 1.11
  • The BRNO catalog was updated from the on-line database (http://var.astro.cz/)
  • Remarks in the Prosper catalog were translated to English.
  • Catalogs Vyškov and Ondøejov were removed from the installation.
Version 1.10
  • New filter based on the days of week
  • The table of star ephemerides shows corresponding days of week.
Version 1.9
  • New filter based on the computation of time of twilight, adjustable solar elevation angle at the twilight.
  • Added support for two new export formats - HTML and CSV.
  • Shows the solar elevation angle in the time of minimum in the info panel.
Version 1.8
  • Shows air-mass coefficient in the information box - at half an hour before the minimum, at time of the minimum and at half an hour after the minimum.

  • Fixed bug in rise time and set time.
Version 1.6
  • Fixed bug in displayed azimuth.

  • Support for multiple locations.
Version 1.4
