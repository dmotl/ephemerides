This is a reduced version of the Hipparchos star catalogue, version 2.

The original data are available here:
https://cdsarc.u-strasbg.fr/ftp/cats/I/196/

The file hip2hd contains two integer numbers separated by an equal sign.
The first number is Hipparcos Input Catalogue running number.
The second number is HD/HDE number.
